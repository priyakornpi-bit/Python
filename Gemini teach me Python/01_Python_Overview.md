# Python Learning Roadmap (จาก W3Schools)

เป้าหมายของชุดเรียนนี้:
- เรียน Python ตั้งแต่พื้นฐาน → ขั้นสูง
- เริ่มจาก Syntax → Data Structures → Functions → OOP → Error Handling
- อธิบาย *ทำไม* ไม่ใช่แค่ *ทำยังไง*
- ยกตัวอย่างโค้ดสั้น ๆ ที่เข้าใจง่าย
- ถามคำถามเชิง Critical Thinking หลังแต่ละหัวข้อ

---

📌 เริ่มที่หัวข้อหลัก:
1) Python Basics  
2) Control Flow  
3) Functions  
4) Data Structures (List / Tuple / Dict / Set)  
5) File Handling  
6) OOP (Class / Object / Inheritance / Polymorphism / Encapsulation)  
7) Error Handling  
8) Modules / Packages  
9) Built-in References
