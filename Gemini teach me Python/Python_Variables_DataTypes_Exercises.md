# คำถามทดสอบระดับ A-Level: Variables & Data Types

## คำสั่ง: จงตอบคำถามต่อไปนี้โดยใช้เหตุผลและความรู้ด้าน Python ให้ละเอียด

1. จงอธิบายว่าทำไมโค้ดต่อไปนี้ถึงเกิดข้อผิดพลาด และจะแก้ไขอย่างไรโดยไม่เปลี่ยนแปลงโครงสร้างหลักของโค้ด

```python
a = [1, 2, 3]
b = a
b.append(4)
print(a)
```

2. จงพยากรณ์ผลลัพธ์ของโค้ดต่อไปนี้ และอธิบายเหตุผลที่ผลลัพธ์เป็นเช่นนั้น โดยเฉพาะกรณีที่มีการเปลี่ยนแปลงค่า

```python
x = 5
y = x
x = 10
print(y)
```

3. จงวิเคราะห์ว่าโค้ดต่อไปนี้ทำงานอย่างไร และผลลัพธ์สุดท้ายเป็นอะไร พร้อมอธิบายความแตกต่างระหว่างการกำหนดค่าและการอ้างอิง

```python

list2 = list1[:]
list1[0] = 99
print(list1, list2)
```

4. จงหาข้อผิดพลาดในโค้ดต่อไปนี้ และอธิบายว่าทำไมถึงเกิดขึ้น รวมถึงวิธีแก้ไขที่ถูกต้อง

5. จงพยากรณ์ผลลัพธ์และอธิบายพฤติกรรมของตัวแปรในกรณีที่มีการใช้ตัวดำเนินการ `is` และ `==`

6. จงอธิบายว่าทำไมโค้ดต่อไปนี้ถึงไม่ทำงานตามที่คาด และเสนอวิธีแก้ไข
